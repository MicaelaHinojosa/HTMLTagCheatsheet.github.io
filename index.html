HTML Tag Cheatsheet

Tag

Name

Description
<body> body The body element defins the document's body.
<p> paragraph Defines a paragraph
<h1> to <h6> heading Defines the heading
<img> image Defines an image.
<br> break Defines a line break.
<b> bold Sets the text to bold.
<strong> strong Selects the text as improtant text.
<i> italic Sets the text to italic text.
<em> emphasize Used to define emphasized text.
